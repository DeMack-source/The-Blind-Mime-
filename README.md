## Site structure

- `index.html` — homepage
- `style.css` — visual design and responsive layout
- `script.js` — mobile navigation and small interactions
- `assets/images` — logo, album covers, and artwork
- `assets/audio` — optional compressed preview audio
- `assets/video` — optional short background or teaser video

## Publish with GitHub Pages

Open repository **Settings → Pages**. Under **Build and deployment**, choose:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

Save the settings. GitHub will generate the public website address.
